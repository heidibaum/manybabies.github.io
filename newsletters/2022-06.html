new project announcement!

MB6: Neonatal Imitation

 
We are extremely excited to announce that ManyBabies has accepted a proposal for a new project on Neonatal Imitation! 

We will be hosting a pair of planning workshops via Zoom to convene and hear presentations from experts on the topic, assemble a diverse leadership team, and start talking about what the project will look like.

More information on planned speakers for each workshop can found via the registration links.

All ManyBabies members are invited to attend.

call for data analysts &
code reviewers


ManyBabies projects offer many opportunities for data analysis contributions. We welcome contributions to data analysis at all levels, from basic code review/error checking to leadership on analysis teams. Expertise in the specific project topic is not required to contribute to data analysis. Working on one of the analysis teams is a great opportunity to become familiar with these large datasets that also provide opportunities for secondary data-anaysis and many-analysts type projects.

Please get in touch if you'd like to contribute to active analysis efforts!

reminders

Report project contributions

If you are contributing to an active MB project, we ask that you fill out your project’s CRediT Form to report your contributions as soon as possible. You will be able to update your contribution report throughout the course of the project, and so responses should reflect actual contributions so far and/or plans to collect data (but not other planned contributions). Contact us with any questions.

ORCiD

As part of our mission to promote open science practices, we strongly recommend that all MB contributors obtain an ORCiD. The sign-up process is fast and easy! We're happy to help.
